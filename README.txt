

This is a Bullet 3D Game created to physics class.

-Controls:
	
	-Car:

	UP arrow 	-> accelerate
	DOWN arrow 	-> decelerate 
	LEFT arrow 	 -> go left
	RIGHT arrow 	 -> go right
	SPACE             -> brake	

	-Tools:

	'2'              -> restart position
	ESCAPE		 -> exit the game
